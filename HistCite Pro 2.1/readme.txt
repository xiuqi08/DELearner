本程序只支持 Windows 操作系统！

选择 Web of Science 的【核心合集】，将需要分析的全部文献按照【全记录与引用的参考文献】形式导出为“.txt”格式（包含500条记录的单个txt文档大小一般是3MB左右，如果你的不是，请仔细查看教程），无需做任何内容修改，直接将这些文件全部放入 TXT 文件夹中，之后双击运行 main.exe 即可自动完成。

您可以使用本程序提供的测试数据【savedress_for_test.txt】来检测该程序在您的电脑上能否正常运行！

由于HistCite内核存在的设计缺陷，默认调用的是系统的IE浏览器（不是edge）来打开网址，而且无法修改。
所以请务必保证你的IE浏览器能够正常工作，例如能够正常打开百度等网站，如果不行请参考这个网址进行修复：
https://jingyan.baidu.com/article/6b97984dc1d9ab1ca2b0bf0c.html

在少部分电脑上会出现导入数据不成功的情况，本软件对此进行了优化，推出了ADV模式，可以解决大部分的兼容性问题。

注意：在ADV模式下不可出现中文路径和中文文件名！你可以将本程序文件夹放到某个盘的根目录下面运行。

2.1版本解决了上个版本偶尔（非常少见）会出现的卡顿问题。

Only Windows operating system is supported!

Export all the infomation from Web of Science Core Collection to txt formate files, and copy them to the path "TXT", then run main.exe and all will be done.
